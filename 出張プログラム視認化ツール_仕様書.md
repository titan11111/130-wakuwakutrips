# 出張プログラム視認化ツール 仕様書

## プロジェクト概要
1ヶ月の出張プログラムをカレンダー形式で視認化し、ドラッグ&ドロップで直感的に地域を配置できるゲーム感覚のWebアプリケーション。最終的にPDF出力が可能。

---

## 1. プロジェクト情報
- **プロジェクト名**：出張プログラム視認化ツール
- **対象プラットフォーム**：iOS（Web/Safari）、一般的なブラウザ
- **言語**：JavaScript（フロントエンド）
- **ゲーム要素**：ドラッグ&ドロップ、アニメーション、色彩設計

---

## 2. ゴール
✅ 1ヶ月の出張プログラムをカレンダーで視認化
✅ ドラッグ&ドロップで地域を配置（将棋のような配置感）
✅ モバイル（iOS）に最適化
✅ PDF出力機能
✅ ゲーム感覚の操作感とワクワク感を実現

---

## 3. 機能要件

### 3-1. カレンダーUI
- **月表示**：指定月（デフォルト：4月2026）の全日程を表示
- **日付セル**：31日分のセル（各月に対応）
- **レイアウト**：7列（日〜土）× 6行の表示
- **セルサイズ**：モバイル（iOS）でタッチしやすいサイズ

### 3-2. ドラッグ可能な地域タグ
**プリセット地域**：
- 熊本
- 福岡1
- 北九州
- 山口
- 東京
- 鹿児島1
- 鹿児島2
- 鹿児島3
- 宮崎
- 久留米

**特徴**：
- 各タグは色分け表示
- 1回ドラッグすると消える（使用済み状態）
- ドラッグ中はタグが半透明になる（ビジュアルフィードバック）

### 3-3. 手入力機能
- **テキスト入力フィールド**：カスタム地域を入力可能
- **追加ボタン**：入力した地域をドラッグ可能リストに追加
- **複数追加対応**：何度でも新規地域を追加可能
- **削除機能**：不要な手入力地域は削除可能

### 3-4. 配置ルール
- **1日1地域**：1つの日付セルには最大1地域のみ配置可能
- **上書き防止**：既に配置されているセルへの新規ドロップはできない（or上書き可能な確認）
- **ドロップエリア**：日付セルがドロップターゲット
- **アニメーション**：配置時にエフェクト（拡大縮小、回転など）

### 3-5. 管理機能
- **配置済み地域の確認**：配置された地域をリスト表示
- **配置のリセット**：カレンダーの内容をリセット
- **アンドゥ機能**：最後の操作を戻す（任意）

### 3-6. PDF出力機能
- **出力内容**：カレンダーと配置されたすべての地域
- **ファイル名**：`出張プログラム_2026年4月_[生成日時].pdf`
- **レイアウト**：A4またはA5サイズ
- **デザイン**：きれいで読みやすいフォーマット

### 3-7. 決定ボタン
- **機能**：最終確認後、PDFを生成してダウンロード
- **配置状態チェック**：最低1つ以上の地域が配置されているか確認
- **エフェクト**：ボタンクリック時に視覚的フィードバック

## 4. UI/UX デザイン要件

### 4-1. ゲーム感覚の実装
- **配色**：カラフルでポップ（地域ごとに異なる色）
- **アニメーション**：
  - タグをドラッグ中：浮遊感
  - ドロップ時：バウンス、光のエフェクト
  - 配置完了時：サクセス音（任意）
- **フォント**：親しみやすく読みやすいフォント
- **背景**：グラデーション、パターンなど工夫

### 4-2. レスポンシブ設計
- **モバイル優先**：iOS Safari での表示を優先
- **タッチ対応**：ドラッグはタッチイベント対応
- **ポートレート/ランドスケープ**：両向き対応
- **画面サイズ**：iPhone SE から iPhone Pro Max まで対応

### 4-3. 視認性
- **情報量の削減**：シンプルで分かりやすいUI
- **ビジュアル階層**：重要な要素を目立たせる
- **コントラスト**：色の組み合わせで操作性を確保

## 7. 操作フロー

### 7-1. 基本フロー
1. ユーザーがページにアクセス
2. カレンダー表示＋ドラッグ可能な地域タグを表示
3. ユーザーが地域タグをカレンダーの日付セルにドラッグ
4. ドロップされた地域が日付セルに配置（タグは消える）
5. 必要に応じて手入力で追加地域を入力・追加
6. すべての配置が完了したら「PDFで決定」をクリック
7. PDF生成＋ダウンロード
8. ユーザーが手動でメール送信

### 7-2. ドラッグ&ドロップの詳細
- **ドラッグ開始**：タグが半透明になる
- **ドラッグ中**：タグが浮遊するアニメーション
- **ドロップ成功**：
  - セルに地域が配置
  - タグは消える（リスト削除）
  - サクセスアニメーション実行
  - 配置済みリストに追加
- **ドロップ失敗**（セルが埋まってる etc）：
  - タグが元の位置に戻る
  - エラーメッセージ表示（任意）

## 8. データモデル

### 8-1. カレンダーデータ
```javascript
{
  month: 4,
  year: 2026,
  days: 30,
  layout: [
    { date: 1, dayOfWeek: 'Tue', location: null },
    { date: 2, dayOfWeek: 'Wed', location: '熊本' },
    ...
  ]
}
```

### 8-2. 地域タグデータ
```javascript
{
  id: 'kumamoto',
  name: '熊本',
  color: '#FF6B6B',
  used: false
}
```

### 8-3. 配置履歴
```javascript
{
  placements: [
    { date: 2, location: '熊本' },
    { date: 5, location: '福岡1' },
    ...
  ]
}
```

## 9. エラーハンドリング

| エラー | 対応 |
|--------|------|
| セルが既に埋まっている | タグを元位置に戻す + メッセージ表示 |
| PDFダウンロード失敗 | ユーザーに通知 + リトライ機能 |
| モバイルでのドラッグ失敗 | Touch API にフォールバック |
